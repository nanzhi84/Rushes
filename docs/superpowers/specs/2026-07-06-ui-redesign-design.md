# Rushes UI 重构设计（2026-07-06）

状态：已经用户确认，进入实施。落地 2026-07-05「UI 抄 ChatCut 布局、时间线保持只读」决策（任务 #29），并按用户新增决策简化素材导入。

## 决策摘要

1. **剪映式两态壳**：首页项目卡片墙 + Case 全屏工作台，常驻左侧项目树退役。
2. **深色剪辑器风**：全局深色（类剪映专业版），Tailwind v4 `@theme` 设计令牌。
3. **素材管理收进工作台**：独立 Project 素材页取消，项目详情与工作台共用同一素材面板组件。
4. **单一本地导入**：素材导入只保留「本地导入」（服务端文件管理器多选文件/文件夹，保留文件夹层级）；上传与 URL 导入不再有 UI 入口（URL 导入走对话由 agent 完成）。

## 1. 两态壳

### 态一 · 首页 `/`

- 顶栏：Rushes 标识 + 连接状态 + 设置。
- 主体：「＋新建项目」+ 项目卡片（封面为项目素材缩略图拼贴、项目名、Case 数/素材数/更新时间；悬浮菜单：重命名/复制/删除）。
- 点击项目卡 → 项目详情 `/projects/:id`：顶部返回 + 项目名；两个 tab——「剪辑 Case」（Case 卡片列表 + 新建剪辑入口，输入目标即建 Case 跳工作台）与「素材」（复用工作台素材面板）+ 设置入口。
- 原 `ProjectHomePage` / `ProjectMaterialsPage` 合并进项目详情；`/projects/:id/materials` 重定向到素材 tab。

### 态二 · Case 工作台 `/projects/:id/cases/:cid`（全屏）

```
┌─────────────────────────────────────────────────┐
│ ⌂  项目A / Case001(可改名)   ● 已连接  [导出] ⚙ │
├──────────────┬───────────────┬──────────────────┤
│              │   素材面板     │    预览播放器     │
│   聊天区      │  (文件夹分组)  │  (纯黑衬底 9:16)  │
│  (可拖宽)     ├───────────────┴──────────────────┤
│              │    时间线（只读通栏，可拖高）        │
└──────────────┴──────────────────────────────────┘
```

- 聊天区默认约 380px 可拖拽调宽；时间线高度可拖拽；面板尺寸存 zustand + localStorage。
- `ProjectTree`、`AppLayout` 现有形态退役；`TreeMutationDialog` 的重命名/删除/复制/移动对话框迁移为通用 `EntityActionDialog` 复用。

## 2. 聊天区（左）

保留现有全部功能只换壳：assistant-ui 消息流（散文流式 + 历史回放）、工具进度条目、Decision 卡片、结构化交互卡、输入框 + 运行状态锁定。Case 成本小计挪到输入框上方细字条。深色气泡风：用户消息右侧浅色块，助手消息无气泡直排，工具进度带图标行内条目。

## 3. 素材面板 ＋ 单一本地导入

- 唯一导入入口「本地导入」：`FsBrowserDialog` 升级为多选文件 + 可选中整个文件夹；选文件夹则递归扫描媒体文件、保留相对路径层级导入。
- 面板：文件夹分组网格（文件夹瓦片 + 素材缩略图瓦片），子文件夹进入 + 面包屑返回；瓦片含缩略图、时长角标、理解状态徽标；点击素材在预览区试看；详情弹层看摘要全文、删除引用。
- 后端配套（唯一后端改动面）：`importLocalMaterial` 支持目录 + 批量；`AssetLink` 增 `rel_dir`（相对所选根目录的子路径）；素材列表 API 返回该字段。走事件溯源单写路径（AssetLinked 事件带字段），contracts + reducer + 迁移同步改。

## 4. 时间线（底部通栏，只读）

- 顶部时间刻度尺 + 当前时码（tabular-nums）。
- 轨道分行：V1 视频（蓝）、A1 音乐（绿）、A2 音效（青）、字幕（紫），行首轨道标签。
- 视频 clip 显示素材缩略图 + 名称；音频 clip 显示 wavesurfer 波形。
- 红色播放头与预览双向联动（播放跟随、点击/拖拽刻度 seek）；缩放控件（滑杆 + 适应全长）；版本历史下拉（切换查看 + 恢复须确认，沿用现有链路）。
- 不做编辑交互；可编辑时间线按原决策推迟到 M9 后。

## 5. 视觉系统（深色剪辑器风）

Tailwind v4 `@theme` 设计令牌，集中在 `index.css`：

- 背景 `#131316`、面板 `#1c1c21`、浮层 `#26262c`、描边 `#2e2e36`、正文 `#ececf1`、次级 `#8f8f9a`。
- 强调色暖橙 `#ff5c38`（主按钮/导出/播放头）；轨道色：视频 `#4a7dd6`、音乐 `#3fa66a`、音效 `#2fa8a0`、字幕 `#8b6fd6`。
- 预览区纯黑 `#000`；圆角 8px 面板；无重阴影靠明度分层；时码 `font-variant-numeric: tabular-nums`。
- 组件只用令牌不写裸色值，为亮色主题留门（本轮不做双主题）。
